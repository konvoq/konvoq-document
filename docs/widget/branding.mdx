# Branding & Appearance 🎨

The Konvoq widget should feel like a natural part of your website — not a foreign object. This page shows you how to make it look exactly the way you want.

Find these settings at **[Dashboard → Widget → Branding](https://app.konvoq.com)**.

---

## What you can customize

| Setting             | What it does                                                      |
| ------------------- | ----------------------------------------------------------------- |
| **Primary color**   | The color of the chat bubble, header bar, and buttons             |
| **Widget icon**     | The image inside the launcher bubble (default is the Konvoq logo) |
| **Widget position** | Bottom-right (default) or bottom-left                             |
| **Launcher size**   | Small, medium, or large — pick what fits your site's layout       |

---

## Choosing your primary color

This is the **most important** branding setting. It changes:

- The launcher bubble background
- The chat window header
- Button colors inside the chat

Pick a color that matches your brand. Paste a hex code (e.g. `#019D91`) or use the color picker.

:::tip Accessibility matters
Make sure there's enough contrast between your primary color and white text. Use a tool like [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) to verify — aim for a ratio of at least 4.5:1.
:::

## Uploading a custom launcher icon

By default the bubble shows the Konvoq logo. Replace it with your own:

1. Go to **Widget → Branding → Icon**
2. Click **Upload Icon**
3. Select your image file

**Accepted formats:** PNG, JPG, JPEG, SVG, WEBP
**Max size:** 2 MB
**Recommended dimensions:** 512 × 512 px (square)

The icon is displayed inside a circle, so avoid putting important content right at the corners.

---

## Widget position

| Option       | Best for                                                             |
| ------------ | -------------------------------------------------------------------- |
| Bottom-right | Most sites — the standard location users expect                      |
| Bottom-left  | Sites where the right side is busy (live chat, cookie banners, etc.) |

---

## Preview your changes

Changes are previewed live in the dashboard before you save. Click **Preview** to open a test widget and see exactly how it looks.
