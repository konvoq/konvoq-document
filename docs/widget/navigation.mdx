# Navigation & Quick Links 🔗

Navigation items are **shortcut buttons** that appear inside the chat window. Think of them like a mini menu — visitors can click them to jump straight to your pricing page, contact form, or docs without typing a word.

Manage these at **[Dashboard → Widget → Navigation](https://app.konvoq.com)**.

---

## What it looks like

When a visitor opens the chat, before they type anything, they see your navigation items as clickable buttons. Each button has a label and opens a URL in a new tab.

---

## Adding navigation items

1. Go to **Widget → Navigation**
2. Click **Add Item**
3. Enter a **Label** — this is what the button says (e.g. `View Pricing`, `Book a Demo`, `Contact Us`)
4. Enter the **URL** — the full link the button opens (e.g. `https://yoursite.com/pricing`)
5. **Drag to reorder** — items appear in the order you set
6. Save

---

## Tips for good navigation items

- **Keep labels short** — 2–4 words max. `Pricing`, `Book Demo`, `Contact Us` work great
- **Use your highest-value pages** — think about what visitors most often want to see
- **Limit to 3–5 items** — too many choices overwhelms visitors
- **Don't duplicate the AI** — don't add a link to a page the AI can already answer questions about

---

## Example setup

| Label       | URL                            |
| ----------- | ------------------------------ |
| Pricing     | `https://yoursite.com/pricing` |
| Book a Demo | `https://cal.com/yourcompany`  |
| Contact Us  | `https://yoursite.com/contact` |
